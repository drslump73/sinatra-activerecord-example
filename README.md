## Features

* Inspired by [ggoral/sinatra-modular-example](https://github.com/ggoral/sinatra-modular-example)
* Modular Style Sinatra Application
* Using ActiveRecord
* Test with: Minitest

## Run server

```bash
$ mkdir -p log
$ mkdir -p tmp/pids
$ mkdir -p tmp/sockets
$ bundle install
$ bundle exec rackup -p 4567
```
